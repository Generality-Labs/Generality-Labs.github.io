A scaffold post exercising the project-wide Quarto setup: shared site chrome from _includes/, shared styles from assets/site.css, and a figure from the gl.js component library.

gl = import("/assets/gl-components/gl.js")

Hello, world. This post exists to prove the shared scaffolding works end to end: the nav, typography, and footer come from the site-wide includes, and the figure below is rendered client-side by an OJS cell calling the house component library.

letters = {
  const counts = {};
  for (const ch of "helloworld") counts[ch] = (counts[ch] ?? 0) + 1;
  return Object.entries(counts).map(([label, value]) => ({ label, value, group: "letter" }));
}
gl.glBars({Plot}, letters, {
  xLabel: "occurrences",
  caption: "Letter frequency in “hello world”.",
  width: 688,
})
